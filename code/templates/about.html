about
peak is a service for auto-generating highlights from twitch vods

how to use
1. access already processed video:
enter/pass the video id into the video_id field
data about the proccessed video is stored for 1 week

2. proccess a new video:
getting code
you can get the code sent to your email by entering/passing your email into the email field
you can get up to 2 codes per day
code is active for 1 day

enter/pass the video id into the video_id field and code into the video_code field
